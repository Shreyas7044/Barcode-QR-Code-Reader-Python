# Barcode-QR-Code-Reader-Python
A Python-based Barcode and QR Code reader using OpenCV, Pyzbar, and Pillow. Captures live video from your webcam, detects barcode/QR patterns, decodes the information, displays it on screen, and saves the decoded text to a file. A great beginner-friendly computer vision and ML project.

This project is a Barcode and QR Code reader built using Python, OpenCV, Pyzbar, and Pillow. It uses your system camera to detect barcodes/QR codes, display the decoded text on screen, and save the result into a text file.
